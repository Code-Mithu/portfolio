# Accessibility Audit Report

## 1. Overview
This report details the accessibility audit of the personal portfolio website, conducted against WCAG 2.1 AA standards.

## 2. Methodology
- **WCAG Review**: Automated and manual verification against WCAG 2.1 AA criteria.
- **Keyboard Navigation Review**: Manual inspection of all interactive elements using `Tab`, `Shift+Tab`, `Enter`, and `Escape` keys.
- **Contrast Review**: Verification of text and UI elements against minimum contrast ratios (4.5:1 for normal text).
- **Screen Reader Review**: Validation of ARIA labels, roles, and focus management during navigation.

## 3. Findings

| Aspect | Status | Observations |
| :--- | :--- | :--- |
| **Semantic HTML** | Pass | Good use of `<main>`, `<nav>`, `<footer>`, and heading levels. |
| **Keyboard Navigation**| Pass | All interactive elements are reachable via `Tab` key; focus management implemented. |
| **Contrast** | Pass | Primary brand colors and text maintain readable contrast ratios. |
| **Screen Reader** | Pass | Main interactive components feature correct ARIA labels and roles. |

## 4. Summary
The portfolio website demonstrates strong adherence to accessibility best practices across all reviewed categories.
